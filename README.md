# zwiki
=====

[中文](https://github.com/jack-zh/zwiki/blob/master/README_zh.md)

A simple wiki(blog?) written in Flask.

## Start

    git clone https://github.com/jack-zh/zwiki.git
    cd zwiki
    pip install -r requirements.txt
    gunicorn app:app
